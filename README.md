# Alshizamin FXD System

Alshizamin FXD System is a client-first, local-first foreign exchange operations PWA designed for fast transaction encoding, currency exchange tracking, operational monitoring, and daily money changer workflows for Alshizamin.

Fast. Local. Reliable FX Operations.

## Features

- Local-first transaction storage
- Offline-capable PWA support
- Fast transaction encoding
- Currency exchange tracking
- Daily totals and operational monitoring
- Income estimation and matching
- Mobile-first responsive interface
- Installable on phones and tablets
- Optimized for real-world money changer operations

## Run

```bash
npm install
npm run dev
```

Build for PWA service worker generation:

```bash
npm run build
npm run start
```
